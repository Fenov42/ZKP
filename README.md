# ZKP
Создание доказательства с нулевой разглашённостью
Инструкция
Скачиваете все три файла 
Разводите файлы prover.py и verifier.py по разным машинам
файл schnorr_core.py должен быть на обеих машинах
запускаете prover.py получаете два файла identity.json это ваш приватный файл его нельзя раздавать
также получаете файл proof.json его отправляете на 2 машину verifier 
запускаете verifier.py указав в файле путь к вашему json файлу 
получаете значение valid или invalid 
параметры кривой для безопасности нужно менять в файле schnorr_core.py
